# Logo

Coloca aqui el logo final de El Rancho.

Nombres recomendados:
- `logo.svg`
- `logo.png`

El sistema intentara mostrar `/brand/logo.svg` primero y usara una marca textual si no existe.

## Imagen de cabecera (pantalla Hoy)

Coloca la foto del encabezado verde aqui con el nombre exacto:

- `hero.jpg`  (ruta final: `public/brand/hero.jpg`)

Recomendado: foto horizontal de la granja, ~1200x500 px, JPG o WebP.
Si el archivo no existe, la pantalla muestra el degradado verde por defecto.
El texto del saludo lleva una capa oscura encima para que siempre se lea.
